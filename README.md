# PROTO_POT_STEPPER_RFID
This Circuit Project was Implemented on Arduino Mega ADK with Protothreads. The Modules used are RC522 RFID, Potentiometer and 28BYJ-48 Stepper Motor with a Motor Driver.

Libraries Included : 
                     - pt.h  
                     - SPI.h 
                     - MFRC522.h 
                     - Stepper.h
